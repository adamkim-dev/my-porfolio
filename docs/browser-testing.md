# Browser Testing

This portfolio is designed to run in Chrome, Safari, and Edge.

## Checklist

- [ ] Load the homepage in Chrome.
- [ ] Load the homepage in Safari.
- [ ] Load the homepage in Edge.
- [ ] Verify dark and light theme toggling works.
- [ ] Ensure the hero and project sections render cleanly on desktop and mobile.
- [ ] Confirm the AI assistant can send a request and display a response.

## Recommended browser checks

- Check the responsive layout at 320px, 768px, and 1200px.
- Validate link interactions and anchor scrolling.
- Confirm readable typography and visual hierarchy.
- Check keyboard focus states for buttons and input fields.
